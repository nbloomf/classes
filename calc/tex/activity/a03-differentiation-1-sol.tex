\begin{enumerate}
\item Compute the following derivative. \[ \frac{d}{dx} \left( 11 x^2 + 8 x + 7 \right) \]

\textbf{Solution:}
Since this function is a polynomial, we can use the power rule on each term. The derivative is $22 x + 8$.
