\begin{enumerate}
\item Compute the following derivative. \[ \frac{d}{dx} \left( 12 x^2 + 12 x + 7 \right) \]

\textbf{Solution:}
Since this function is a polynomial, we can use the power rule on each term. The derivative is $24 x + 12$.
