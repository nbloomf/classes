\begin{enumerate}
\item Compute the following derivative. \[ \frac{d}{dx} \left( 13 x^2 + 10 x + 13 \right) \]

\textbf{Solution:}
Since this function is a polynomial, we can use the power rule on each term. The derivative is $26 x + 10$.
