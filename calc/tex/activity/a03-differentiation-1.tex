\begin{enumerate}
\item Compute the following derivative. \[ \frac{d}{dx} \left( 10 x^2 + 5 x + 9 \right) \]

\textbf{Solution:}
Since this function is a polynomial, we can use the power rule on each term. The derivative is $20 x + 5$.
